# Library Management System

## Tech Stack

### Backend

* Java
* Spring Boot
* MySQL
* JPA/Hibernate
* Swagger OpenAPI

### Frontend

* Angular
* TypeScript
* Bootstrap

---

## Features

### Book Management

* Create Book
* Update Book
* Delete Book
* Search Books
* View Books

### Member Management

* Create Member
* Update Member
* Deactivate Member
* View Members

### Issue / Return Workflow

* Issue Book
* Return Book
* Transaction History
* Stock Management

---

## Project Structure

library_management

* library_api
* library-ui

---

## Backend Run

```bash
mvn spring-boot:run
```

Swagger:

```text
http://localhost:8080/swagger-ui.html
```

---

## Frontend Run

```bash
ng serve
```

Frontend:

```text
http://localhost:4200
```
